# hbox

 - in LongUI, This is equivalent to the [box](./box.md) element.
 - LongUIにて、この要素は [box](./box.md) 要素と同等です
 - LongUI 中, 本元素等价于[box](./box.md)


*HOW IMPL*

```cpp
  // h-box layout
  using UIHBoxLayout = UIBoxLayout;
```

## Example

```xml
<!-- Two button on the right -->
<hbox>
  <spacer flex="1"/>
  <button label="Connect"/>
  <button label="Ping"/>
</hbox>
```
